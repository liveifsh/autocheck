система проверки.
Для работы необходимо устоновить pandas
$pip install pandas.

settings.py - файл с настройками.
INPUT_DIR - указывается путь к папке с вводам к тестам
OUTPUT_DIR - указывается путь к папке в которой лежат файлы с этононным выводом
USERS_DIR - папка, в которой находятся все пользователи
SUL_NAME - названия файла с решением, оно должно лежать в папке конкретного юзера
RESULT_FILE - файл для вывода таблички с результатом
SLEEP_TIME - время между проверками


Как создавать тесты.

В папке INPUT_DIR, должны лежать файлы с расширением .txt
в первой строке указывается тип проверки, количество баллов за задачу, индекс строки проверки (если проверяются все строки -1), через точку с запятой
Тип проверки либо FULL - баллы выстовляются только если пройдены все тесты
либо PART - баллы выстовляются равномерно за каждый под тест
количество баллов целое число.


Каждый тест начинается с = с отдельной строки.
Далее идет содержание файла на конкретный тест.

В папке OUTPUT_DIR содеражаться должны лежать файлы с расширением .txt
Имена файлов должны совпадать с именем того файла, к проверке которого они относятся.
Каждый тест начинается с = с отдальной строки.
Далее идет содержание файла на конкретный текст.
