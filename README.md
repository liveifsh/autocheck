# Система проверки заданий.
Мяу

## Установка зависимостей: 
```pip install -r requirements.txt```

## Настройка
settings.py - файл с настройками.
INPUT_DIR - папка с тестами
OUTPUT_DIR - папка с эталонным выводом тестов
USERS_DIR - папка, в которой находятся все пользователи
SUL_NAME - название файла с решением, оно должно лежать в папке конкретного юзера
RESULT_FILE - файл для вывода таблички с результатом
SLEEP_TIME - время между проверками


## Как создавать тесты.

В папке INPUT_DIR, должны лежать файлы с расширением .txt
в первой строке указывается тип проверки, количество баллов за задачу, индекс строки проверки (если проверяются все строки -1), через точку с запятой
Тип проверки:
- FULL - баллы выстовляются, только если пройдены все тесты
- PART - баллы выстовляются равномерно за каждый под тест

Количество баллов - целое число.

Каждый тест начинается с символа '=' с новой строки.
Далее идет содержание файла на конкретный тест.

В папке OUTPUT_DIR содеражаться должны лежать файлы с расширением .txt
Имя файла должно совпадать с именем того файла, к проверке которого он относится.
Каждый тест начинается с символа '=' с новойстроки.
Далее идет содержание файла на конкретный тест.
